Reconocimiento de Poses

Detecta y clasifica posiciones del cuerpo humano usando la cámara en tiempo real

🧠 ¿Cómo funciona?

El modelo usa PoseNet para detectar los puntos clave del cuerpo (articulaciones, cabeza, manos) y luego clasifica la postura según las clases entrenadas en Teachable Machine.

📁 Cómo subir tu modelo

  • 1Exporta tu modelo desde teachablemachine.withgoogle.com → Export Model → Download my model.
  • 2Descomprime el ZIP y renombra la carpeta a pose_model.
  • 3En GitHub: Add file → Upload files → sube la carpeta completa.
  • 4Haz commit y espera 1 minuto para que GitHub Pages se actualice.

Reconocimiento de Imágenes

Clasifica objetos o imágenes capturadas por la cámara en tiempo real

🧠 ¿Cómo funciona?

El modelo de imágenes analiza cada frame de la cámara y lo compara con las clases que fueron entrenadas. Devuelve una probabilidad para cada clase definida.

📁 Cómo subir tu modelo

Reconocimiento de Audio

Detecta y clasifica comandos de voz o sonidos en tiempo real usando el micrófono

🎙️

🧠 ¿Cómo funciona?

El modelo de audio usa la Transformada de Fourier (FFT) para analizar las frecuencias del micrófono y clasifica el sonido según las clases entrenadas, con un umbral de probabilidad del 75%.

📁 Cómo subir tu modelo

  • 1Exporta tu modelo desde teachablemachine.withgoogle.com → Export Model → Download my model.
  • 2Descomprime el ZIP y renombra la carpeta a audio_model.
